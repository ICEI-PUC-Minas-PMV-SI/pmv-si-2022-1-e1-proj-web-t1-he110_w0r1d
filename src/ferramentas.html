Ferramentas para Prototipagem

Figma [Gratuito]

Figma é uma plataforma de design que busca trazer funcionalidades de design poderosas que você já ama e trazer um fluxo de trabalho mais eficiente. Algumas das funcionalidades incluem:

  • Uma moderna ferramenta de caneta: Desenhe em qualquer direção com a Rede de Vetor.
  • Ferramenta Arc: crie relógios, telas ou gráficos com a ferramenta Arc.
  • Seja expressivo com o OpenType: Apenas clique nas funcionalidades avançadas de fonte para expressar em cheio a marca.

Axure RP [Estudante]

Para estudantes o Axure RP oferece uma licença gratuita, o software é potente e oferece diversos modelos de elementos já prontos para a criação do mockup, apesar de ter uma curva de aprendizado, sua interface é amigável.

Para grandes projetos o Axure RP oferece eficiência na hora de organizar o material e tornar o desenvolvimento mais flúido possível.

Como usar o postman?

Postman [APIs]

O Postman é uma ferramenta que dá suporte à documentação das requisições feitas pela API. Ele possui ambiente para a documentação, execução de testes de APIs e requisições em geral.

Ao utilizá-lo, você passará a trabalhar com APIs de modo mais eficiente, construindo solicitações rapidamente e, ainda, poderá guardá-las para uso posterior, além de conseguir analisar as respostas enviadas pela API.

Um bom motivo para usar essa ferramenta é que, por meio dela, é possível reduzir drasticamente o tempo necessário para testar e desenvolver APIs.

New Relic Tutorial [Inglês]

New Relic [APM]

A função primária do New Relic já é muito interessante. O software ajuda no monitoramento de aplicativos e sites em tempo real, tanto ao controlar o desempenho como determinar se a experiência do usuário está sendo positiva.

Por exemplo, a solução pode determinar quanto tempo o banco de dados demora a responder um determinado pedido. Dependendo do tempo, pode ser avaliado se é rápido ou lento e, caso não esteja na velocidade ideal, é possível avaliar se é um problema pontual ou algum problema relacionado a qualquer componente ligado ao banco de dados.

Ferramentas para Colaboração e Planejamento

Trello [Gratuito]

O Trello é uma ferramenta de gerenciamento de projetos gratuita e online. Disponível para desktop e também para celulares Android e iPhone (iOS), a plataforma é extremamente versátil e pode ser usada tanto para o acompanhamento de tarefas pessoais quanto para organizar projetos que envolvem equipes numerosas em grandes empresas.

Com uma interface intuitiva e recursos como checklists, upload de arquivos multimídia e etiquetas coloridas, o Trello pode ser um aliado útil no planejamento de estudos, viagens e atividades de trabalho.

Veja ao vídeo completo da Sabrina Adriane sobre o Trello.

Slack [Freemium]

O Slack é um app de mensagens para empresas que conecta as pessoas às informações de que elas precisam. Reunindo pessoas para trabalhar como uma equipe unificada, o Slack transforma a forma como as organizações se comunicam.

O Slack oferece suporte para um trabalho assíncrono. Quando o trabalho é organizado em canais, você pode acessar as informações de que precisa no momento que quiser, qualquer que seja sua localização, fuso horário ou função. Faça perguntas, veja as novidades e compartilhe atualizações sem precisar seguir a agenda de outras pessoas.
