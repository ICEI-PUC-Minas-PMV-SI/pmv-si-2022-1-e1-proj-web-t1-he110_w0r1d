O que é e como surgiu a linguagem Python?

O que é Python?

Python é uma linguagem de programação extremamente simples e versátil, pois, como sua sintaxe é moderna e objetiva, ela foi elaborada para que as desenvolvedores escrevam instruções com menos linhas de código.

No caso, por ser de fácil compreensão, foi uma linguagem que ganhou popularidade entre as empresas devido ao seu código enxuto. Além disso, ela possui uma grande quantidade de bibliotecas disponíveis para auxiliar no nosso dia a dia, e tem uma grande comunidade de desenvolvedores.

Como surgiu a linguagem Python?

Python foi desenvolvido por Guido van Rossum no final dos anos 80 e início dos anos 90 no Instituto Nacional de Pesquisa em Matemática e Ciência da Computação na Holanda. Seu objetivo era criar uma linguagem de script mais eficiente, capaz de acessar recursos do sistema operacional Amoeba.

Desde seu lançamento em 1994, a linguagem Python recebeu diversas atualizações. Ela tem o código aberto e é disponibilizado sob a licença PSF License Agreement, que é semelhante à General Public License(GPL). Atualmente, está na versão 3.10.4 e é possível fazer o download gratuitamente a partir da página oficial da linguagem (https://www.python.org/).

Curiosidade: Apesar de seu logotipo ser o de duas cobras entrelaçadas, o nome escolhido não tem nenhuma relação com a espécie de cobras de mesmo nome. A escolha foi em função de um programa de televisão da época criado por um grupo de comediantes chamado Monty Python.

Porque Python é tão valorizado?

A linguagem Python ocupa hoje, o segundo lugar na preferência mundial. Existem diversos fatores que contribuem para esse resultado positivo. Confira por que vale a pena aprender a programar nessa linguagem:

Fácil de aprender: Python tem poucas palavras-chave, estrutura simples e uma sintaxe claramente definida. Isso permite que o aluno aprenda o idioma rapidamente.

Fácil de ler: O código Python é mais claramente definido e visível aos olhos.

Fácil de manter: O código-fonte do Python é muito fácil de manter.

Uma ampla biblioteca padrão: a maior parte da biblioteca do Python é muito portátil e compatível com várias plataformas em UNIX, Windows e Macintosh.

Modo interativo: Python tem suporte para um modo interativo que permite testes interativos e depuração de trechos de código.

Portátil: Python pode ser executado em uma ampla variedade de plataformas de hardware e possui a mesma interface em todas as plataformas.

Extensível: Você pode adicionar módulos de baixo nível ao interpretador Python. Esses módulos permitem que os programadores adicionem ou personalizem suas ferramentas para serem mais eficientes.

Bancos de dados: Python fornece interfaces para todos os principais bancos de dados comerciais.

Escalável: O Python fornece uma estrutura e suporte melhores para programas grandes do que o shell script.

A maneira mais fácil de executar o Python

A maneira mais fácil de executar o Python é usando o Thonny IDE.

O Thonny IDE vem com a versão mais recente do Python incluída. Portanto, você não precisa instalar o Python separadamente.

Siga as etapas a seguir para executar o Python em seu computador:

  • Baixe o Thonny IDE .
  • Execute o instalador para instalar o Thonny em seu computador.
  • Vá para: Arquivo > Novo. Em seguida, salve o arquivo com .pyextensão. Por exemplo, hello.py, example.py, etc.
    Você pode dar qualquer nome ao arquivo. No entanto, o nome do arquivo deve terminar com .py
  • Escreva o código Python no arquivo e salve-o.
  • Em seguida, vá para Executar > Executar script atual ou simplesmente clique em F5 para executá-lo.
